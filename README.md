# RareBox
**一个完全开源免费的WearOS手表和Android调试工具箱。麻雀虽小五脏俱全！**

![RareBox](https://github.com/Genouka/RareBox/assets/142009020/90a9719c-9f8e-4d96-a1a2-4c9b7902841b)
   
# 功能
在手表上自调试：

- [x] adb终端
- [x] 安装应用
- [x] 修改dpi、显示范围

用Android设备调试手表：

- [x] adb终端
- [x] 安装应用
- [x] 修改dpi、显示范围
- [x] 传输文件

用手表调试另一手表：

与用Android设备调试设备是一回事。

用手表/Android设备调试另一Android设备：

没考虑过，目前调试Android8以下设备应该没有问题。
